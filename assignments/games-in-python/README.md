# 📘 Assignment: Games in Python

## 🎯 Objective

Build a Hangman game in Python to practice core programming concepts such as loops, conditionals, string handling, and user input.

## 📝 Tasks

### 🛠️ Build the Hangman Game Loop

#### Description
Create the main game flow for Hangman. The program should choose a hidden word, repeatedly ask the player for letter guesses, and update the game state after each guess.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list.
- Prompt the player to guess one letter at a time.
- Display the current progress of the word using underscores for unknown letters.


### 🛠️ Handle Win and Loss Conditions

#### Description
Add logic to track incorrect guesses and determine when the game ends. Provide clear feedback for both successful and unsuccessful outcomes.

#### Requirements
Completed program should:

- Track the number of incorrect guesses remaining.
- End the game when the word is fully guessed or attempts are exhausted.
- Display a clear win or lose message at the end of the game.
